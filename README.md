# BMI Checker (C)

A simple C program to calculate Body Mass Index (BMI) and classify health status.

## Features
- Takes user input for weight (kg) and height (cm).
- Calculates BMI using the formula: BMI = weight / (height/100)^2
- Classifies results into categories:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
- Lightweight and fast (written in C).
- Runs in any standard terminal.

# BMI Checker (C)

یک برنامه‌ی ساده به زبان C برای محاسبه‌ی شاخص توده‌ی بدنی (BMI) و نمایش وضعیت سلامتی.

## قابلیت‌ها
- دریافت وزن (کیلوگرم) و قد (سانتی‌متر) از کاربر
- محاسبه‌ی BMI با فرمول: BMI = وزن / (قد/100)^2
- دسته‌بندی نتایج در گروه‌های:
  - کم‌وزن
  - وزن نرمال
  - اضافه‌وزن
  - چاق
- سبک و سریع
